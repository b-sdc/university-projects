# Temple University Incident and Budget Analysis
## Applied Statistics 3503-2022

This repo contains the code that supports the following research:

This project examines initiatives that influence crime in areas surrounding the university through analysis of the university’s campus safety budget. Crime incident counts were analyzed over a 35-month period alongside corresponding annual budget data, using [open data](https://opendataphilly.org/datasets/crime-incidents/)
 provided by the Philadelphia Police Department. The analysis focuses on incidents involving fraud, narcotic/drug violations, theft, and vandalism/criminal mischief within Police District 22, which includes Temple University’s Main Campus and nearby areas with a high concentration of off-campus student housing.
